# 🎓 TARUMT 考勤应用程序

一款简单美观的桌面应用程序，专为 TARUMT 学生设计，可快速记录考勤并管理多个账户。

> 📖 **[English Version (英文版)](README.md)**

---

## 📥 安装

### 下载与安装

1. **下载适合您平台的应用程序：**

   - **macOS**：下载 `TARUMT Attendance.dmg` 或 `.zip` 文件
   - **Windows**：下载 `.exe` 安装程序
2. **安装应用程序：**

   - **macOS**：打开 DMG 文件并将应用程序拖到应用程序文件夹
   - **Windows**：运行安装程序并按照设置向导操作
3. **启动应用程序**：从应用程序文件夹（macOS）或开始菜单（Windows）启动
4. **首次安全警告：**

   **macOS 用户**：如果看到安全警告：

   - 右键点击应用程序 → 选择"打开"
   - 在对话框中点击"打开"
   - 您只需执行一次此操作

   **Windows 用户**：如果看到 "Windows 已保护你的电脑"：

   - 点击 **"更多信息"**（小文字链接）
   - 点击 **"仍要运行"** 按钮
   - **应用程序是安全的** - 这对于免费/学生项目来说是正常的

## 📖 用户指南

### 🔐 首次登录

1. **打开应用程序** - 您会看到登录界面
2. **输入您的凭据：**
   - 学生 ID（例如：`2500001`）
   - 密码
3. **点击"登录"**
4. **完成！** 您的账户会自动保存

> 💡 **提示：** 下次打开应用程序时，您只需点击您的名字即可立即登录！

---

### 👥 多账户管理

该应用程序支持多个学生账户 - 非常适合共享设备或管理多个账户的情况。

#### **切换账户：**

1. **注销后**，您会看到账户选择界面
2. **点击任何已保存的账户** 即可立即登录（无需密码！）
3. 所有账户都会显示姓名和学生 ID

#### **添加更多账户：**

1. 在账户选择界面上，点击 **"添加新账户"**
2. 输入新的学生 ID 和密码
3. 点击"登录" - 新账户将添加到您的列表中

#### **删除账户：**

1. 在账户选择界面上，找到要删除的账户
2. 点击账户旁边的 **垃圾桶图标** 🗑️
3. 该账户将从您保存的列表中删除

#### **登录期间：**

如果您想在登录页面切换账户：

- 点击 **"选择账户"** 按钮返回账户选择界面

---

### ✅ 记录考勤

这是主要功能 - 超级简单！

1. **登录后**，您会看到考勤界面
2. **讲师在投影仪/屏幕上显示考勤代码**
3. **在输入框中输入代码**（通常是 4-6 位数字）
4. **点击"提交考勤"**
5. **成功！** 您会看到确认消息

> ⚠️ **重要：** 您必须连接到 TARUMT 校园 WiFi 才能正常工作！

---

### 📋 考勤历史

应用程序会自动记录您今天提交的所有考勤记录。

#### **您会看到的内容：**

提交考勤后，"今日考勤"部分会显示一张卡片，内容包括：

- ⏰ **时间** - 记录时间（例如："上午 10:30"）
- 🔢 **代码** - 您使用的考勤代码
- 📚 **课程代码** - 例如："BMIT2203"
- 📖 **课程名称** - 例如："人机交互"
- 🏫 **课堂详情** - 时间和教室（例如："上午 10:00 - 中午 12:00，DK 4"）
- 👨‍🏫 **讲师** - 授课教师

#### **功能：**

- ✅ 提交新考勤时自动更新
- ✅ 仅显示今天的所有考勤记录
- ✅ 即使注销后再登录也会保留（同一天）
- ✅ 每天午夜自动重置

## 🛡️ 高级技巧：避免被检测

> ⚠️ **注意：** 这些技巧适用于担心系统检测的用户。大多数用户不需要这些。

### 📱 使用自定义设备信息

默认情况下，应用程序使用您计算机的唯一设备 ID。如果您想使用不同的设备信息（例如您的手机）：

1. **点击设置图标** ⚙️（注销按钮旁边）
2. **查看自动检测到的设备信息：**
   - 设备 ID
   - 设备型号
   - 主机名
3. **输入自定义值：**
   - 您可以输入 **手机的设备 ID** 和 **设备型号**
   - 这会使提交看起来像是从手机而不是电脑提交的
4. **点击"保存设置"** 应用
5. **恢复默认：** 点击"重置为自动检测"

#### **为什么要使用这个？**

- 如果您希望考勤提交看起来像是从移动设备提交的
- 有助于避免可能标记来自同一台计算机的多个账户的模式

#### **在哪里获取手机设备 ID：**

**安卓（Android）：**

- 设置 → 关于手机 → 版本号（点击 7 次启用开发者模式）
- 设置 → 开发者选项 → 设备 ID

**苹果（iPhone）：**

- 设置 → 通用 → 关于本机 → 型号
- 对于设备 ID，使用手机的序列号或 IMEI

---

### 🌐 多账户使用不同 IP 地址

如果您从同一设备管理多个账户，学校系统可能会检测到不同学生账户使用相同的 IP 地址。

#### **解决方案：在账户之间重新连接 WiFi**

1. **为账户 1 提交考勤**
2. **从应用程序注销**
3. **断开 TARUMT WiFi 连接：**
   - macOS：点击 WiFi 图标 → 关闭 WiFi
   - Windows：点击 WiFi 图标 → 断开连接
4. **等待 5-10 秒**
5. **重新连接到 TARUMT WiFi**
6. **使用账户 2 登录**
7. **提交考勤** - 现在您有了不同的 IP 地址！

#### **为什么有效：**

- 当您重新连接到 WiFi 时，路由器会为您分配一个新的 IP 地址
- 这使得两个账户看起来像是从不同的设备/位置登录的
- 减少系统标记来自同一设备的多个账户的可能性

#### **重要说明：**

- ⏰ 每个账户需要额外 10-20 秒
- 🔄 仅在担心基于 IP 的检测时才需要
- 💡 与自定义设备 ID 结合使用时最有效

## ⚠️ 要求

### 仅限校园 WiFi

**您必须连接到 TARUMT 的校园 WiFi 网络。** 考勤系统将拒绝来自以下来源的提交：

- ❌ 家庭 WiFi
- ❌ 移动数据或热点
- ❌ 公共 WiFi
- ❌ VPN 连接

### 支持的平台

- ✅ macOS（Intel 和 Apple Silicon）
- ✅ Windows 10/11

## 🔒 隐私与安全

- ✅ 所有数据 **仅存储在您的设备本地**
- ✅ 无第三方服务器或跟踪
- ✅ 密码仅发送到 TARUMT 的官方身份验证服务器
- ✅ 计算机上的每个用户配置文件都有单独保存的账户
- ✅ 应用程序不收集或共享任何个人信息

## 🐛 故障排除

### "用户 ID 或密码无效"

- ✅ 仔细检查您的凭据
- ✅ 先登录 TARUMT 门户网站验证密码
- ✅ 确保您连接到校园 WiFi

### "代码无效" 或 "不允许注册此课程"

- ✅ 与讲师确认代码
- ✅ 检查您是否已注册该课程
- ✅ 确保您连接到校园 WiFi
- ✅ 代码可能已过期 - 获取新代码

### 保存的账户消失了

- 账户是按设备上的用户配置文件保存的
- 检查您是否使用了不同的用户账户
- 只需再次登录 - 您的账户将被保存

### 应用程序无法打开（macOS）

- 右键点击 → "打开"（仅首次）
- 或：系统设置 → 隐私与安全性 → "仍要打开"
